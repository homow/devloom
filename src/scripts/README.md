# ⚠️ Warning: Development-Only Scripts

This folder contains scripts **for development and testing only**.

## Important Notes:

- Do **not** run these scripts on production servers.
- Scripts here can create privileged users or modify your database.
- Only trusted developers should execute these scripts locally.
- For example, `create-super-admin.ts` will create a SUPER_ADMIN user in the database.

**Always review scripts before running them.**
